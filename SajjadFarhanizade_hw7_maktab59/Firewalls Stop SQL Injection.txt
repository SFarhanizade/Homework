علت اینکه فایروال‌های رایج مانند فایروال ویندوز  نمی‌توانند جلوی SQL Injection را بگیرند این است که این فایروال‌ها اغلب Packet Filter هستند و فقط بر اساس آدرس آی‌پی بسته‌های ارسالی تصمیم‌گیری می‌کنند. برای جلوگیری از SQL Injection باید از فایروال‌هایی استفاده کرد که سطح دسترسی Application داشته باشند؛ مانند WAF (Web Application Firewall). این فایروال‌ها این نوع حملات را تشخیص می‌دهند و می‌توانند جلوی آنها را بگیرند.
